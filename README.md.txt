# SmartGuard: Система предиктивного мониторинга умного дома

Программный комплекс для обнаружения сетевых аномалий в IoT-сетях на основе алгоритма Random Forest. Система анализирует поведенческие паттерны трафика и оповещает об угрозах в реальном времени.

## 🛠 Технологический стек
* **Язык:** Python 3.12
* **ML-библиотеки:** Scikit-learn, Pandas, Joblib
* **Сетевой протокол:** MQTT (paho-mqtt)
* **Интерфейс:** Tkinter (Desktop GUI)
* **База данных:** SQLite

## 🚀 Инструкция по запуску

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/твой-логин/название-репозитория.git
   cd название-репозитория

1. Установите необходимые библиотеки:
code
Bash
pip install -r requirements.txt

2. Настройка уведомлений (опционально):
Отредактируйте файл SmartGuard.py, вставив свой Telegram Token в переменную TELEGRAM_TOKEN.

3. Запуск мониторинга:
Запустите основной файл приложения:
code
Bash
python SmartGuard.py

4. Тестирование системы:
Для имитации атак запустите симулятор в отдельном окне терминала:
code
Bash
python test_attack.py

📂 Структура проекта
SmartGuard.py — основное приложение с графическим интерфейсом.
test_attack.py — скрипт симуляции сетевого трафика и атак.
anomaly_detector.pkl — предобученная модель машинного обучения.
scaler.pkl — параметры нормализации данных.
final_home_iot_dataset_v4.csv — датасет для верификации.
